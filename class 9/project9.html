Typography

A font family is a group of typefaces that share a common design but vary in styles such as weight(bold, light), width (condensed, expanded), or slant (italic, oblique).

The border is a CSS property that defines the boundary and style (color, width, pattern) of an element, while the border-radius property allows you to round the corners of the element.







The transform: rotate() CSS function rotates an element around a fixed point (typically its center) by a specified angle in degrees (e.g., rotate(45deg)), where positive values rotate clockwise and negative values rotate counterclockwise.




The text-decoration CSS property controls the appearance of text, allowing you to add or remove effects like underlines, overlines, line-throughs, or set their style and color.

The text-shadow CSS property adds a shadow effect to text, allowing you to specify the shadow's horizontal and vertical offset, blur radius, and color.




The box-shadow CSS property adds shadow effects to an element's box, allowing you to set horizontal and vertical offsets, blur radius, spread radius, and color for the shadow.

The text-indent CSS property specifies the amount of horizontal space (indentation) applied to the first line of text in a block or paragraph.
